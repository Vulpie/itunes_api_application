
# iTunes Music Search

Sample app using iTunes API to search for songs by artist or title

[Live example](https://music-search-api-app.herokuapp.com/)

---

## Installation

Install required dependencies

```bash
    npm install
```

Build bundle with webpack

```bash
    npm run build
```

Run application

```bash
    npm run start
```
